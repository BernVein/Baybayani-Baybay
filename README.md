# Baybayani-Baybay

E-commerce platform for Baybayani.

## Status

Work in progress. Some features and layouts may change.

## Tech Stack

- Vite
- React
- HeroUI
- Tailwind CSS
- TypeScript
- Supabase

## Features

### Customer

- Shop browsing
- Cart management
- Order tracking
- User profile and settings
- Messaging

### Admin

- Dashboard summary
- Order management
- Product management
- User management
- Messaging

## Development

### Install

```bash
npm install
```

### Run

```bash
npm run host
```
